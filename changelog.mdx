## How to Check Your Version

Follow these steps to verify your installed version of godfred osei.

<Steps>
  <Step title="Run Version Command" icon="terminal">
    
````bash
godfred-oisei --version
````

    This displays the current version, such as `v1.2.0`.
  </Step>
  <Step title="API Endpoint" icon="api">
    
    Use the version endpoint:
    
````javascript
const response = await fetch('https://api.example.com/v1/version');
const data = await response.json();
console.log(data.version); // "v1.2.0"
````
  </Step>
  <Step title="Update" icon="download">
    
````bash
npm update godfred-oisei
````

    Pull the latest release automatically.
  </Step>
</Steps>
